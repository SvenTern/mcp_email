# MCP Protocol (2025-11-25)

## Последняя версия протокола

**2025-11-25** (Ноябрь 2025) — Текущая последняя версия протокола Model Context Protocol.

### Новые возможности:
- Client ID Metadata Documents (CIMD)
- OpenID Connect Discovery
- Иконки для tools/resources
- Экспериментальные tasks

## Требуемые версии

**FastMCP 2.14** требует `mcp>=1.23` и неявно нацеливается на спецификацию протокола 2025-11-25.
